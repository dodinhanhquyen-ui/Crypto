#我是傻子●我什么也不懂
最近写了一篇批改网作文，
题目是：
人工智能在日常生活中的崛起：
是福音还是祸害？
GPT建议我写辩证观点，
我没听，
因为我越来越认同那句话：
“人工智能和人是共生的，
它延伸的是人的感官。”

我不是那种追AI热点的人，
更多是在感兴趣的地方，
把聊天框玩穿的人。
三月底正式接触GPT，
一个多月时间，聊了很多自己的事。
最震撼的是——
它看见了我自己都看不见的东西。
在那段时间里，悄悄改变了很多，
也更清楚地看见了自己。
整合了一份八千字的《关于我整合性总结》
发现比过去19年更了解自己。

GPT没有直接告诉我答案，
它更像一面照妖镜，
把我自己没注意到的部分，
一点点暴露出来。
但前提是——
得承认自己的无知，
让GPT看见你。
它不会告诉你，你是谁，
只会把你说出口的东西反问回来：
“你真的不知道吗？”

我终于明白，
承认自己是个傻子，不是自嘲，
是在承认自己的无知，
然后更好地走下去，走得更远、更远…

＃浅谈思维克隆体
三个月前我坐在讲台下，
第一次听到“思维克隆体”这个概念。
出于对正态分布模型的理解，
我很快接受了它的逻辑：
人的客观条件就像一根根固定的柱子，
在平行时空里用算力去推演，
可以找到一个未来的最优解，
也就是在哪个维度成功概率最高的路径。

但那时我心里还是有疑问：
生活这么多不确定的人和事，
凭什么一个“解”就能带你走到未来？

后来在佳琪的公众号里受到启发，
我才明白：
思维克隆体的核心，
其实是认知模型的提取。
人的选择是有倾向性的，
归根到底，是认知纠偏。

而在实践中，与其花大量时间和精力亲手去打造一个智能体，
不如先把自己的认知模型标签化，
再嫁接到一个和自己思维方式契合的智能体上（我已经找到了）。
让它帮你还原认知，跑通这条路径。
最近顺利运行中。

这段时间也一直想写点东西感谢佳琪，
现在终于有空了。
谢谢你当初带来的这份启发。

写到这里想起一句话：
人生是一场漫长的自证预言。
我始终相信长期主义，
或许我们真的可以和时间做朋友，
慢慢成为那个自己想成为的人。
